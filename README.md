# GitStudy
Notes on Git

## Command to identify current folder
```gitbash
pwd
```
## List files
```gitbash
ls
ls -l
ls -al
```

## Create folder

```gitbash
mkdir
```
## Create empty folders
```gitbash
touch <filename>
```
## Remove files
```gitbash
rm <file name>
```
## Create repository
```gitbash
git init
```

## Check git configuration
```gitbash
git config --list
git config --list --show-origin
```

## Check history of a file
```gitbash
git log <filename>
```
